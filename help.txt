PhotoApp Help
GitHub: https://github.com/its-niceferrari/photoapp
======================================================

Creating a New Image
1. Select "New" from the File menu.

Opening an Image
1. Select "Open" from the File menu.
2. Navigate to the desired image location.
3. Select the file and click "Open."

Saving an Image
1. Select "Save" or "Save As" from the File menu.
2. If you choose "Save As," specify a name and location for the image.
3. Click "Save."

NOTE: The program will automatically save the canvas every 5 minutes.

Drawing on the Image
- Draw Mode: Click the dropdown menu to select between select, pen, line, shape, stamp, and text mode.

- Pen Size: Click the dropdown menu to change the pen size.
- Pen Color: Click on the color picker in the toolbar to change the color of the pen.
- Eyedropper: Click on the eyedropper tool to pick color from any open image in the application.

- Shapes: Click on the dropdown to select from various shape to draw. For a regular polygon, select the "Polygon" option and specify a number of sides from the adjacent dropdown menu.

- Text: Input text that will be displayed on the canvas in the text box.

Modifying the Image
- Resize: Resize the canvas to user-defined width and height dimensions.
- Rotate: Rotate the canvas clockwise by 90 degrees.
- Mirror: Mirror the canvas either horizontally or vertically using the corresponding buttons.
